Scheduler application was made to ease a scheduling of the appointments for the users of global organization,
and conduct business in multiple languages and time zones.

Author: Samir Cokic.
Contact: scokic1@wgu.edu
Application Version: 1

IDE: IntelliJ IDEA 2021.3.3 (Community Edition)
JDK version: java version "16.0.1" 2021-04-20
JAVAFX version: javafx-17.0.1


DIRECTIONS:
User enters username and password. If those were correct the main window with available appointments opens.
On the left side of the screen are controls are used to add, update or cancel appointment, as well as
to add, update or delete customer.
The main part of the window consists of tabs. Main tabs are Appointments, Customers and Reports.
The appointments tab consists of three sub tabs that show all appointments or the weekly or monthly appointments.
The Customers tab show customers.
The Reports tab shows reports in two separate tabs: Customer Appointments and Contact's Schedule.
Customer appointments tab show three reports: Customer Appointments By Month, Customer Appointments By Type and
Customer Appointments By Contact.



Description of the additional report in part A3f:
Customer's Appointment By Contact report shows the number of appointments that each customer had with every
contact.

MySQL Connector driver: mysql-connector-java-8.0.30
